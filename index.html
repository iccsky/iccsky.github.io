MarkdownPad快捷键

Bold(加粗) Ctrl+B

Emphasize(斜体) Ctrl+I

Blockquote(引用) Ctrl+Q

Code(代码) Ctrl+K

H1(h1标题) Ctrl+1

H2(h2标题) Ctrl+2

H3(h3标题) Ctrl+3

Link(链接) Ctrl+L

Image(图片) Ctrl+G

Unordered List(无序) Ctrl+U

Ordered List(有序) Ctrl+Shift+O

Timestamp(时间) Ctrl+T

Horizontal Rule(分割线) Ctrl+R

熟悉这些快捷键的话，基本上不用到鼠标了，可以专注码字了。好了，MarkdownPad就介绍到这里。

常用hexo命令

Veröffentlicht am 2018-01-11 | Edited on 2018-01-13 | in Hexo

常见命令
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

缩写：
hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy

组合命令：
hexo s -g #生成并本地预览
hexo d -g #生成并上传

远程部署
打开git bash，输入hexo d就会将本次有改动的代码全部提交
